# Glossary – Monthli

- **Portfolio** – Collection of positions held at a broker, with base currency and snapshot settings.
- **Position** – Holding in a single instrument (equity/ETF/crypto), defined by symbol and/or ISIN.
- **Transaction** – Buy/sell/dividend/cash movement imported from DEGIRO CSV.
- **PriceTick** – Daily price observation for a symbol/ISIN.
- **MonthlySnapshot** – Aggregated monthly portfolio value and performance metrics (MoM, YTD, etc.).
- **DEGIRO Portfolio CSV** – Export with current holdings (Product, Symbool/ISIN, Aantal, Slotkoers, Waarde in EUR).
- **DEGIRO Transactions CSV** – Export with all cash & trade movements over time.
